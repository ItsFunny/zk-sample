
## requirements

* x86
* `go` <https://go.dev/doc/install>
* `docker` <https://www.docker.com/get-started>
* `docker-compose` <https://docs.docker.com/compose/install/>

## init

```bash
git submodule init
git submodule update
```

## build

```
./build.sh
```

## run

```
./run.sh
```

## stop

```
docker-compose stop
```

## restart

```
./restart.sh
```

## down

```
docker-compose down
```
